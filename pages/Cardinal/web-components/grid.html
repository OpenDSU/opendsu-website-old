Every web-site is trying to have a very nice layout of elements inside the page and also to to be responsive. Psk-grid is intending to do so. The scope of the component is to set up the layout according to a minimal setup: number of columnsand the layout rules.

This component is dividing the inner content into columns and applies the layour rules, both set up within the columns and layout attributes.

As a reference we used the bootstrap documentation: bootstrap grid.

It is very important to keep in mind that if the number of columns is not the same with the number of the values for a layout rule, that rule will be ignored. The bolow examples will show how this component works.

The last example for this component will show what will happen if the values are not according to the rules for layout attribute.

This component will render the complete text between the tags in the same number of columns as the property.
