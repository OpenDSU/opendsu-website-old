The mount() method adds a reference to another dossier at a specific path. mount(path, archiveIdentifier, callback) path The path where the dossier's reference is added.

 The mount() method creates a reference to a dossier and keeps it in a special file stored inside the dossier that keeps the reference at the path "/manifest". If a reference with the same name and at the same path already exists, an error is thrown. The mounted dossier can store references to other dossiers. As a result, all mounted dossier form a tree of arbitrary depth. Inside the tree, a dossier is uniquely identified by the path that leads to its reference. For example, let's say that a dossier having the SEED "seed2" is mounted in a dossier with the SEED "seed1" at path "/foo/bar" which, in turn, is mounted inside the dossier whose SEED is "seedRoot" at path "/dir/baz". The SEED "seed2" that uniquely identifies a dossier can be retrieved by traversing the path "/dir/baz/foo/bar".
